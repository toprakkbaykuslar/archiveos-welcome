ArchiveOS Welcome 🚀
ArchiveOS Welcome is a modern onboarding application designed to greet users upon their first boot, providing essential ecosystem information and quick access points.

In line with the ArchiveOS philosophy, it aims to deliver a minimalist, stable, and user-friendly first impression.

✨ Features
Modern Interface: A clean and responsive GUI built using PyQt6.

System Information: Displays the current system version and basic resource status.

Minimalist Design: Follows the "Clean & Stable" aesthetic to match the overall system environment.

🛠️ Installation & Usage
To run this application, you must have Python and pip installed on your system.

1. Clone the Repository
git clone https://github.com/archiveos-owner/archiveos-welcome
cd archiveos-welcome
2. Install Dependencies

Install the required PyQt6 library using pip:
pip install PyQt6

4. Launch the Application
python main.py

🎨 Design Standards
ArchiveOS Welcome utilizes the Horizon design language. It avoids unnecessary visual clutter and focuses on directing the user to the information they need immediately.
